# Projeto clone login Facebook

Este projeto foi desenvolvido visando clonar a página de login do Facebook.

## Página clonada

https://www.facebook.com/?locale=pt_BR

## Clone do projeto

https://github.com/RaphaelDamico/clone-facebook-login-react.git

## Executar projeto

npm run dev
